Project Procedural Geometric Modeling

3D Graphics

Project Overview

This project demonstrates procedural geometric modeling through code. The demo generates several core 3D meshes, including planes, cubes, spheres, tori, cylinders, and cones, by constructing vertex data, index buffers, normals, and topology directly in C++.

Tasks Completed

  • Implemented procedural mesh generation for multiple 3D primitives.
  • Created reusable mesh-building functions using vertices, normals, texture coordinates, and index buffers.
  • Integrated the generated meshes into an interactive graphics demo.
  • Created a dedicated portfolio page to showcase the assignment with screenshots and project information.

Reflection

This assignment helped me understand how vertex data, indices, normals, and topology work together to create reusable procedural shapes. It also improved my understanding of how low-level mesh data is organized before being rendered in a real-time graphics pipeline.
